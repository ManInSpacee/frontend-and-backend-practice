# frontend-and-backend-practice

Основные технологии:
- Frontend: JavaScript, HTML, CSS
- - Инструменты: npm/yarn, Git

Краткий запуск (пример):
1. Клонировать репозиторий:
   git clone https://github.com/ManInSpacee/frontend-and-backend-practice.git
2. Перейти в нужную папку (frontend/backend) и установить зависимости:
   npm install
3. Запустить локально:
   npm start

Примечание: это учебные задания, в рамках которых было реализовано два сайта на фреймворке Bootstrap и чистый HTML/CSS.
